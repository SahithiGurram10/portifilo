Currency Converter App

This Currency Converter App allows users to convert between different currencies with real-time exchange rates. The app is built using Python and utilizes APIs to fetch the latest exchange rates, ensuring accurate conversions. It includes a user-friendly interface where users can input the amount and select the currencies for conversion. The app also displays historical exchange rate data, providing insights into currency trends over time.
